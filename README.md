# Symfony New

This is a Symfony-based web application skeleton for new project.

## Installation

### Prerequisites

- PHP >=8.2
- Make (for Windows : https://gnuwin32.sourceforge.net/packages/make.htm)
- Composer
- Symfony CLI

### Setup

1. Clone the repository:

    ```bash
    make sfnew-clone
    ```

2. Name the repo

3. Composer install
   
   ```bash
    composer install
    ```
